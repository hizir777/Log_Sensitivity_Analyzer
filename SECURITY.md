Log Sensitivity Analyzer is a security-focused project. Please follow

## Reporting a Vulnerability

If you discover a security issue, **do not open a public GitHub issue**.
Instead, report it privately via email:

- **Contact:** security@your-domain.example

Please include:

- A clear description of the issue
- Steps to reproduce
- Affected components/files
- Any suggested mitigations

We aim to acknowledge reports promptly and provide a remediation timeline
after confirming the issue.
